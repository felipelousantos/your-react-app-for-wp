# Your React App for WP

## Installation

Things that you might want to do:

- search/replace `your_react_app_for_wp` and `your-react-app-for-wp` to the text domain and prefix of your plugin
- change from `your-react-app-for-wp.php` to the plugin file name that you would like
- search/replace `Your React App for WP` to the name of your plugin
